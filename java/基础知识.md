<h3 id = "1">数据类型<h3>
两大数据类型：  
1、内置的基本数据类型  
2、引用数据类型 
<h4>1、内置数据类型<h4>
分为四大类：整数型、浮点数型、字符型、布尔型  
整数型：byte,short,int ,long  
 byte:  
  8位，有符号  
  -128（-2^7）--- 127（2^7-1）;   
 
 short:  
 16位  
 
 int：   
 32位，常用  
 
 long:  
 64位  
 例子： long a = 100000L，Long b = -200000L。后缀大小都可以，但是小写l容易混淆，强制大写。  

浮点数型:float,double  
float：  
单精度，32位  

double：  
双精度，64位  

字符型：char  
单一的 16 位 Unicode 字符；  
char 数据类型可以储存任何字符；  

布尔型：boolean  
 表示一位的信息；只有true/false  

<h4>2、引用数据类型<h4>
对象，数组。
